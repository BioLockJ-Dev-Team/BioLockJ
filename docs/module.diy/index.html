DIY package

The DIY package allows users to customize BioLockJ to fit their needs.

GenMod

#BioModule biolockj.module.diy.GenMod

Description: Allows user to add external scripts into the BioLockJ pipeline.

Options:

  • genMod.launcher
  • genMod.param
  • genMod.scriptPath
